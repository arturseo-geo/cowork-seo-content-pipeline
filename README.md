# SEO Content Pipeline Plugin for Claude Cowork

**Keyword research → SERP brief → GEO draft → rank tracking. One pipeline, zero context switching.**

Built by [Artur Ferreira](https://thegeolab.net) | The GEO Lab

---

## Commands

| Command | What It Does |
|---------|-------------|
| `/seo:research [keyword]` | Opportunity score, volume, KD, SERP features, cluster map |
| `/seo:brief [keyword]` | SERP-feature-targeted brief with PAA questions and entity targets |
| `/seo:draft [keyword or brief]` | Full GEO-optimised WordPress-ready HTML draft |
| `/seo:audit` | Content decay audit — refresh / merge / redirect / leave |
| `/seo:ranks` | Weekly rank report with title rewrite detection |

## Agent

`content-pipeline-run` — Full autonomous Sunday pipeline: validate keyword → brief → draft → quality gate → deliver to Drive + log to Notion.

## Connectors

DataForSEO · Google Search Console · Google Analytics 4 · Firecrawl · Notion · Google Drive
